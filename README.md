# Pokedex Guide

A fun and interactive Pokédex web app that lets you explore all 151 original Pokémon — powered by the PokéAPI, built with pure Vanilla JavaScript.

# Features

🔍 Real-time search — filter Pokémon by name as you type
🎨 Dynamic color-coded cards based on Pokémon type
📦 Fetches live data from PokéAPI for all 151 original Pokémon
⚡ Animated search bar toggle
🏋️ Displays base experience, weight, and type for each Pokémon
🖼️ Official Pokémon sprites from PokeAPI



# Technologies Used

HTML5
CSS3
Vanilla JavaScript (ES6)
PokéAPI — free and open Pokémon data API
Font Awesome — icons
Google Fonts — Sniglet font

# What I Learned
Fetching data from a public REST API using fetch and async/await
Dynamically creating and injecting DOM elements with JavaScript
Real-time filtering with input event listeners
CSS animations and toggle effects with classList
Mapping API data to dynamic UI components
